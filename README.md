# The Sharpe Sports Report 🎙️

An interactive, high-impact web platform dedicated to NFL Draft analysis, player breakdowns, and community sports debate. This site serves as the official digital home for the *Sharpe Sports Report* podcast.

## 🚀 Live Demo
You can view the live site here: [The Sharpe Sports Report](https://github.io)

## ✨ Key Features
- **The Huddle:** A modern hero section with fluid typography and glassmorphism effects.
- **Latest Episodes:** An interactive grid showcasing recent podcast episodes with direct links to Spotify.
- **Community Engagement:** Integrated forms via Formspree for listeners to share ideas or apply to be a guest.
- **Accessibility Focused:** Designed with WCAG standards in mind, including semantic HTML5, ARIA labels, and high-contrast styling.

## 🛠️ Built With
- **HTML5:** Semantic structure for better SEO and accessibility.
- **CSS3:** Custom properties (variables), Flexbox/Grid layouts, and radial-gradient glow effects.
- **Google Fonts:** Utilizing 'Oswald' for a bold sports aesthetic and 'Open Sans' for readability.
- **Formspree:** Backend handling for community form submissions.

## 📂 Project Structure
- `index.html`: The core structure and content of the site.
- `styles.css`: Custom styling, including fluid design and dark-mode aesthetics.
